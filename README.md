# üìö Bookshelf App

A React-based web application for managing your personal book collection. Users can add, edit, search, delete, and move books between **Not Finished Reading** and **Finished Reading** sections. All data is stored locally using localStorage, ensuring your collection persists even after closing or refreshing the app.

## üîó Demo

Demo: https://iamstarck.github.io/bookshelf-app/

## ‚ú® Features

- **Add Book**: Add a new book with its title, author, and publication year.
- **Edit Book**: Update details of an existing book.
- **Move Book**: Toggle a book between **Not Finished Reading** and **Finished Reading.**
- **Delete Book**: Remove books permanently.
- **Search Books**: Find books by title.
- **Local Presistence**: Data is saved using `localStorage`, so nothing is lost during page reloads.

## üîß Technologies Used

- **React (TypeScript)**: UI and app structure
- **Zustand**: Global state management.
- **React Hook Form + Zod**: Form handling and validation.
- **ShadCN + Tailwind CSS**: Component styling and layout.
- **localStorage**: Browser-based data persistence.
